# Rclone плагин для Double Commander

[In english](README.md)

Это плагин для файлового менеджера [Double Commander](https://doublecmd.sourceforge.io), он может работать с Double Commander в операционных системах Windows, macOS и Linux. 

В операционной системе Windows этот плагин так же совместим с файловым менеджером [Total Commander](https://www.ghisler.com).

Плагин позволяет файловому менеджеру Double Commander (или Total Commander) взаимодействовать с файлами в облачных хранилищах используя программу rclone. [Rclone](https://rclone.org) является программой для командной строки, позволяющей работать с облачными хранилищами (например, такими, как Google диск, Yandex диск, Dropbox и т.д.). С технической точки зрения этот плагин является оберткой вокруг программы rclone.

## Как установить и настроить

### 1. Установить и настроить Rclone:

Примеры как можно установить и настроить Rclone в разных операционных системах.

#### В операционной системе Windows

1. Скачать Rclone с их официального сайта https://rclone.org , распаковать и поместить в любую удобную папку на компьютере, например, *C:/rclone/*
2. В папкe с программой Rclone открыть командную строку, выполнить команду `./rclone.exe config` и следуя подсказкам настроить облачные хранилища.

#### В операционной системе macOS

1. Удобнее всего установить Rclone используя [homebrew](https://brew.sh) : `brew install rclone` .
2. В командной строке выполнить `rclone config` и следуя подсказкам настроить облачные хранилища.

#### В операционной системе Linux

1. Установить Rclone через пакетный менеджер вашей версии Linux, например, для дистрибутивов основанных на Debian: `sudo apt install rclone` .
2. В командной строке выполнить `rclone config` и следуя подсказкам настроить облачные хранилища.

### 2. Добавить плагин в Double Commander (или Total Commander)

#### Инструкция для Double Commander

1. Скачать плагин (файл *.wfx64) со страницы релизы (Releases) этого репозитория или получить этот файл, скомпилировав плагин из исходного кода.
2. В программе Double Commander открыть окно настроек: ***Настройки > Параметры*** 
3. В окне настроек в боковом меню выбрать пункт ***Плагины WFX***
4. Нажать кнопку ***Добавить*** и указать путь к файлу плагина (файл с расширением **.wfx64**).

#### Инструкция для Total Commander

1. Скачать плагин (файл *.wfx64) со страницы релизы (Releases) этого репозитория или получить этот файл, скомпилировав плагин из исходного кода.
2. В программе Total Commander открыть окно настроек: ***Конфигурация > Настройки*** 
3. В окне настроек в боковом меню выбрать пункт ***Плагины***
4. В пункте ***Плагины файловой системы(.WFX)*** Нажать кнопку ***Настройка***, далее, в новом окне нажать кнопку ***Добавить*** и указать путь к файлу плагина (файл с расширением **.wfx64**).

### 3. Настроить плагин

1. В Double Commander открыть вкладку **//** (**Открыть список VFS**). На вкладке **vfs:** открыть папку **Rclone**. Аналогично в Total Commander из выпадающего меню выбрать пункт **Сеть / FS-плагины**, далее, открыть папку **Rclone**.

Далее, пункт 2 зависит от опеационной системы. Примеры настройки плагина в разных операционных системах.

#### В операционной системе Windows

2. В коренной папке плагина, дважды кликнуть (открыть) пункт с названием **&lt;Settings&gt;**. Далее дважды кликнуть пункт с названием **&lt;0_edit_rclone_executable_binary_path&gt;** и в открывшемся окне написать путь к программе rclone, например, `C:\rclone\rclone.exe` и нажать кнопку **Ok**.

#### В операционной системе macOS

2. В коренной папке плагина, дважды кликнуть (открыть) пункт с названием **&lt;Settings&gt;**. Далее кликнуть пункт с названием **&lt;0_edit_rclone_executable_binary_path&gt;** и в открывшемся окне написать путь к программе rclone, например, `/opt/homebrew/bin/rclone` (если rclone устанавливался через homebrew) и нажать кнопку **Ok**.

#### В операционной системе Linux

2. Если вы устанавливали rclone через пакетный менеджер, то можете пропустить этот пункт. В противном случае, в коренной папке плагина нужно дважды кликнуть (открыть) пункт с названием **&lt;Settings&gt;**. Далее кликнуть пункт с названием **&lt;0_edit_rclone_executable_binary_path&gt;** и в открывшемся окне написать путь к программе rclone и нажать кнопку **Ok**.

Следующий пункт одинаковый для всех операционных систем.

3. Вернуться обратно в коренной каталог плагина (дважды кликнув по пункту **..**). Если все настроено верно, то рядом с пунктом **&lt;Settings&gt;** должен отображаться список облачных хранилищ (в виде отдельных папок), настроенных ранее в Rclone. 

## Как использовать

В Double Commander открыть вкладку **//** (**Открыть список VFS**). На вкладке **vfs:** открыть папку **Rclone**. Аналогично в Total Commander из выпадающего меню выбрать пункт **Сеть / FS-плагины**, далее открыть папку **Rclone**.

Если ранее все было настроено верно, то в коренном каталоге плагина должен отображаться список облачных хранилищ (в виде отдельных папок), ранее настроенных в Rclone. 

## Для разработчиков

Инструкция отличается в зависимости от операционной системы.

### В Windows

Для компиляции плагина использовался компилятор MSVC. Чтобы скомпилировать плагин нужно:

1. Установить [Visual Studio Build Tools]( https://visualstudio.microsoft.com/ru/downloads/?q=build+tools)
2. Запустить скрипт компиляции:
    ```
    ./compile.bat
    ```

### В macOS

Для компиляции плагина использовался компилятор clang++, входящий в состав Xcode Command Line Developer Tools. Чтобы скомпилировать плагин нужно:

1. Установить Xcode Command Line Developer Tools:
    ```
    xcode-select --install
    ```
2. Отметить скрипт компиляции как исполняемый файл и запустить:
    ```
    chmod +x compile.sh
    ./compile.sh
    ```

### В Linux

Для компиляции плагина использовался компилятор clang++. Чтобы скомпилировать плагин нужно:

1. Установить компилятор clang через пакетный менеджер вашей версии Linux, например, для дистрибутивов основанных на Debian: 
    ```
    sudo apt install clang
    ```
2. Отметить скрипт компиляции как исполняемый файл и запустить:
    ```
    chmod +x compile.sh
    ./compile.sh
    ```
